# קבצי אודיו למשחק

תיקייה זו צריכה להכיל את קבצי הסאונד הבאים:

## קבצים נדרשים:
- `correct.mp3` - צליל לתשובה נכונה (בסיסי)
- `wrong.mp3` - צליל לתשובה שגויה
- `boing.mp3` - צליל קפיצה
- `victory.mp3` - צליל ניצחון (חגיגי)
- `cheer.mp3` - צליל תשואות
- `sparkle.mp3` - צליל נצנוץ/קסם

## איפה למצוא קבצי סאונד חינמיים:
1. **Freesound.org** - ספריית סאונד חינמית
2. **Zapsplat.com** - אפקטי קול חינמיים
3. **Mixkit.co** - סאונד אפקטים בחינם
4. **Pixabay** - גם סאונדים

## טיפ מהיר:
אם אין לך קבצי אודיו כרגע, האפליקציה תעבוד גם בלעדיהם (עם warning בקונסול).
