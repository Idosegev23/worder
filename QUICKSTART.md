# 🚀 התחלה מהירה - WordQuest

## שלב 1: התקנת תלויות

```bash
npm install
```

## שלב 2: הרצת האפליקציה

```bash
npm run dev
```

האפליקציה תרוץ על: **http://localhost:5173**

## שלב 3: כניסה לאפליקציה

### משתמש חדש:
1. לחץ על "הרשמה"
2. מלא שם פרטי + שם משפחה + סיסמה
3. בחר אווטאר
4. התחל לשחק!

### אדמין:
1. לחץ על "כניסת אדמין"
2. Username: `אילנית שגב`
3. Password: `123456`

---

## 🎮 מה תראה?

### למשתמשים:
- **מסך התחברות** → הרשמה פשוטה
- **בחירת אווטאר** → סגנונות מגוונים של DiceBear
- **בחירת קטגוריה** → 4 קטגוריות (שמות עצם, פעלים, מילות יחס, שמות תואר)
- **משחק** → תרגום מילים + **אפקטים משוגעים**!
  - תשובה נכונה → 65% סיכוי לאפקט חגיגי (קונפטי, זיקוקים, וכו')
  - תשובה שגויה → תמיד אפקט שובב (מסך מתהפך, כפתור בורח, וכו')
- **מתנות** → בחירה בין 2 מתנות בסיום

### לאדמין:
- **ניהול מילים** → CRUD מלא, תרגומים אלטרנטיביים
- **ניהול משתמשים** → רשימת משתמשים + **הצגת/עריכת סיסמאות**
- **התקדמות תלמידים** → סטטיסטיקות מפורטות לכל תלמיד (ניסיונות, % הצלחה, פירוט תשובות)
- **ניהול מתנות** → הוספה/עריכה/מחיקה
- **גיבוי ושחזור** → ייצוא/ייבוא JSON
