# Changelog - Worder Game

## [2025-11-21] - פישוט למיתר

### 🎯 שינויים עיקריים:
- **פירושים מרובים → פירוש יחיד**: כל מילה עכשיו עם פירוש אחד בלבד (לא צריך את כל הפירושים כדי לצדוק)
- **תרגומי משפטים**: פישוט המשפטים להתאמה לרמת כיתה א'
- **משחקים ישנים**: הוספת Pronouns ו-Vocabulary (6 קטגוריות)
- **חלוקת Am/Is/Are**: חולק ל-2 קטגוריות עם עירוב חיובי ושלילי
- **Have/Has**: רק משפטים חיוביים (ללא שלילה ושאלות)
- **כפתור השמעה**: רק במשחקי תרגום (לא במשחקי השלמת משפטים)

### דוגמאות לשינויים:
- **לפני**: "careless: רשלני, חסר אחריות" → **אחרי**: "רשלני"
- **לפני**: "You can adjust the chair to make it higher or lower" → **אחרי**: "אפשר להתאים את הכיסא"
